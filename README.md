streamlit programs repository, various programs to learn streamlit with the aim to make useful frontend components using python only
>_learning and creating projects from freecodecamp.org and youtube tutorials_

### To run the programs in the repo
```bash
python -m venv .venv
source .venv/bin/activate.fish
pip install streamlit #and other needed packages (maybe numpy, matplotlib, pandas, etc.)
streamlit run app.py
```
